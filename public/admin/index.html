Whole-Site Access Password

The entire dashboard is locked behind this single password — anyone who wants to view it (players, staff, guests) must enter it first. Change it any time to instantly cut off access for anyone who currently has the old password. Share the new one only with people you want to be able to view the site right now.

Game Day Tab Password

A second password that locks the Game Day tab specifically — even people with the whole-site password need this to open it. Useful for keeping live Force Rating entry restricted to coaches on game day.

The password used to log into this admin dashboard. Changing it will sign out everyone currently logged in (including you) — you'll need the new password to log back in.

Opposition Scouting

Build a player profile for each opposition team. Pick a team, then add/edit their players with stats from the latest KPI summary PDF (KE%, M, HBG, LBG, CP, UP, R50, HR) and a photo URL from their WAFL profile page. The position you choose (Key/Gen Def, Mid, Mid-Fwd, Ruck, Gen/Key Fwd) determines whether a player shows up under the Defenders, Midfielders or Forwards tab on the dashboard. Saving replaces that team's whole roster — so just paste in the latest numbers each week before you play them.
